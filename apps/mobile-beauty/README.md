# CafeTrack Mobile Beauty

App móvil para gestionar **citas y turnos** de:
- Barberías
- Peluquerías
- Salones de belleza

## Funcionalidades iniciales
- Configuración del negocio (dueño + tipo de negocio).
- Registro de turnos/citas.
- Cola activa de turnos.
- Cambio de estado del turno (pendiente → confirmada → en_servicio → finalizada).

## Scripts
- `npm run start`
- `npm run android`
- `npm run ios`
