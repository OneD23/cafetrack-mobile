# Roadmap por fases - Super App

## Fase 1 (MVP inicial)
**Pedidos + negocios + delivery**
- onboarding básico de usuarios y negocios
- catálogo de productos
- creación y gestión de pedidos
- delivery básico (asignación/estado)
- apps base: cliente + backend

## Fase 2
**Tracking + chat + wallet**
- tracking en tiempo real de órdenes
- chat cliente-negocio-repartidor
- wallet base (saldo, movimientos, pagos simples)

## Fase 3
**Marketplace + servicios**
- publicaciones de marketplace
- checkout marketplace
- vertical de servicios bajo demanda (agenda/solicitudes)

## Fase 4
**Escalado y pagos avanzados**
- optimización multi-ciudad/multi-país
- pagos avanzados (split, conciliación, antifraude)
- observabilidad avanzada y hardening operativo

## Notas de decisión abierta
- Definir estrategia exacta de geolocalización y proveedor de mapas.
- Definir pasarela de pagos principal por país/región.
- Decidir si backend migra gradualmente a TypeScript o mantiene JavaScript con tipado por paquetes compartidos.
